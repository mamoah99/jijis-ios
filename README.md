# Jiji's Patisserie iOS App

A SwiftUI preorder app for a micro-bakery, built as a portfolio project.

## Features
- View weekly menu
- Product detail page with image carousel
- Quantity selector
- Cart management
- Pickup slot selection
- Tip selection
- Mock payment flow
- Order confirmation
- Google Sheets + Apps Script prototype backend
- Current orders entry point

## Tech
- Swift
- SwiftUI
- MVVM
- Google Sheets + Apps Script
- URLSession
- async/await

## Status
Prototype / portfolio project in active development.

## Notes
This app uses a lightweight Google Sheets backend for demo purposes and is structured so the backend can later be swapped to Firebase or Supabase.
